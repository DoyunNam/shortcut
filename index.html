Shortcut : Mac version

Brave

  • 새로고침 : command + R

Mac

  • move desktop : ctrl + left/right arrow

Excel

  • 피벗 테이블 새로고침 : 피벗 테이블에 마우스 오른쪽 클릭 후 새로 고침 클릭

Visual Studio Code

  • open and close side bar : command + B
  • 라인 첫 부분 또는 끝 부분으로 커서 이동 : command + left or right arrow
  • 단어 단위로 커서 이동 : option + left or right arrow
  • 라인 전체 선택 : command + L
  • 주석 : command + /
  • 라인 잘라내기 : command + X
  • 라인 복사하기 : command + C
  • 라인 위 또는 아래로 이동 : option + up or down arrow
  • 커서 삽입 : option + click
  • 위에 한 줄 추가 : command + enter
  • 밑에 한 줄 추가 : command + shift + enter
  • 커서 위치에 주석 추가 : shift + option + A
  • 같은 단어 반복 선택 및 수정 (한번씩 누를 때마다 커서가 하나씩 추가됨) : command + D
참고 홈페이지 : 1 2 3
